# TMM Chat Bot

This is a chatbot application built using Python.

## Features
- Conversational chatbot
- AI based responses
- API integration

## Technologies Used
- Python
- FastAPI
- Machine Learning
- NLP

## Installation

git clone https://github.com/Premyarramreddy/tmm_chat_bot.git
cd tmm_chat_bot
pip install -r requirements.txt

## Run the project

python app.py
