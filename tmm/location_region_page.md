Telugu Matchmakers Location Page

The Location page collects where the bride or groom is currently living. The user must select the living country, state, and city. The available living countries are India, United Kingdom, United States, Canada, and Australia.

If the selected country is India, the user must enter a pincode. If the selected country is United Kingdom, United States, Canada, or Australia, the user must enter a zip code.

When the selected living country is NOT India, the following additional fields are shown and become mandatory:
Visa Status
Country of Citizenship
Birth Country

If the selected living country is India, these three fields are hidden and are not required.

This means a user will only see Visa Status, Country of Citizenship, and Birth Country after selecting United Kingdom, United States, Canada, or Australia as the living country.

When the user is on the Location page, the profile completion is ten percent. After the user fills the mandatory fields and clicks Continue or Save and Complete Later, the profile completion becomes twenty percent and the user is taken to the Religion and Community page.

If the user clicks the Back Arrow, the user is taken to the Bride or Groom Name page if applicable, and the profile completion remains ten percent. If the data was not saved previously, the entered data is not stored.

The Continue button is enabled only when all mandatory fields are filled. Clicking Continue validates the required fields, saves the data, moves to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default and becomes enabled only when all required details are entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.

If a user asks why Visa Status is not visible, the correct explanation is that Visa Status only appears when the living country is selected as United Kingdom, United States, Canada, or Australia. If the country is India, Visa Status is not shown.
