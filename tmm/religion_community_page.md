Telugu Matchmakers Religion and Community Page

The Religion and Community page collects the religious, cultural, and marital details of the bride or groom.

The user must select the following mandatory fields:
Religion
Community
Sub Community
Preferred Languages
Marital Status

Gothram is optional.

If the user selects that they have children, the user must also enter how many children they have. If the user selects that they do not have children, the number of children field is not required and will not be enforced.

When the user is on the Religion and Community page, the profile completion is twenty percent. After the user fills all mandatory fields and clicks Continue or Save and Complete Later, the profile completion becomes thirty percent and the user is navigated to the Profession and Education page.

If the user clicks the Back Arrow, the user is navigated to the Location page. The profile completion remains twenty percent. If the user did not previously save the data, the entered information is not stored.

The Continue button is enabled only when all mandatory fields are completed. Clicking Continue validates the required fields, saves the data, moves the user to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default and becomes enabled only when all mandatory fields are entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.

If a user asks why the number of children is required, the correct explanation is that it becomes mandatory only when the user selects that they have children.
