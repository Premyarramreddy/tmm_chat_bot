Telugu Matchmakers Profile Picture Page

The Profile Picture page allows users to upload photos for their profile. All uploaded photos are subject to verification.

The user must upload a minimum of three photos and a maximum of six photos. The photos must be clear, visible, and unique.

When the user is on the Profile Picture page, the profile completion is ninety percent. After the user uploads the required number of photos and clicks Continue or Save and Complete Later, the profile completion becomes one hundred percent and the profile setup is completed.

If the user clicks the Back Arrow, the user is navigated to the About page. The profile completion remains ninety percent. If the user did not previously save the data, the uploaded photos are not stored.

The Continue button is enabled only when at least three photos are uploaded. Clicking Continue validates the photos, saves the data, completes the profile setup, and moves the user forward.

The Save and Complete Later button is disabled by default. It becomes enabled only when at least three photos are uploaded. When clicked, it saves the uploaded photos, exits the profile setup flow, and redirects the user to the Home Page.
