1. Overview

The Chat Module allows users to communicate with mutually connected profiles using:

Text messages

Voice recordings

Audio calls

Video calls

Media sharing (images & videos)

Chat is available only after both users accept a connection request.

Users can access Chat via the Chat icon in the bottom footer.

2. Access Rules
Condition	Result
User not logged in	Chat cannot be accessed
No connections	Chat list will be empty
Mutual connection exists	Chat features enabled
User blocked or reported	Chat disabled
3. Chat Listing Page

After tapping the Chat icon, users see the Chat Listing Page.

Displays:

Connected profiles only

Last message preview

Last activity timestamp

Unread message indicator

If the list is empty, the user has no active connections.

If chat features are visible but not working, the user is connected and the issue is feature-related.

4. Chat Window

Opens when a user selects a connected profile.

Components:

Profile header

Message display area

Message input box

Media & call controls

5. Chat Features
Text Messaging

Send/receive text

Emoji support

Voice Messages

Record and send voice notes

Playback available

Audio Calls

One-to-one audio calls

Call history stored

Video Calls

One-to-one video calls

Supports front and rear camera

Media Sharing

Send images and videos

Preview before sending

Profile Access

Users can view the connected person's full profile from the chat header.

6. User Controls (⋮ Menu)
Feature	Description
Block User	Stops all communication permanently
Report User	Sends report to moderation and blocks the user
Clear Chat	Removes messages from current user view only

Blocked or reported users cannot reconnect automatically.

7. Privacy & Security

Chats are private and encrypted

Users cannot access other users’ chats

Media requires device permission

Screenshots/downloads follow app policy

8. Data Storage

Stored data includes:

Messages

Media references

Timestamps

Call logs

Users can clear chats from their view, but this does not delete the other user's copy.

9. Chat Feature Issue Causes (Specific Reasons)

If a user is already connected but chat features fail, the issue may be due to:

Text / Message Issues

Poor internet connection

App not updated

Server delay

Voice Recording Issues

Microphone permission not enabled

Background noise suppression conflict

Low device storage

Audio/Video Call Issues

Camera or microphone permission denied

Weak network or unstable Wi-Fi

Other apps using camera/mic

VPN or firewall blocking call service

Media Upload Issues

Large file size

Unsupported file format

Low storage space

Slow internet

These are feature-level problems, not connection issues.

10. System Error Responses
Scenario	System Action
No Internet	Show offline message
User Blocked	Disable chat
Connection Removed	Close chat
Media Upload Failed	Show retry option
Call Permission Missing	Prompt to enable permission
