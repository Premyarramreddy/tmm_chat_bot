Telugu Matchmakers Family Details Page

The Family Details page collects family and address information of the bride or groom. Family address information is not public.

The following fields are mandatory:
Mother Name
Father Name
Sisters
Brothers
Country
Address Line 1
State
City
House status 
Family profession/occupation

The following fields are optional:
Address Line 2

If the selected country is India, Pincode is mandatory.
If the selected country is United Kingdom, United States, Canada, or Australia, Zip Code is mandatory.

When the user is on the Family Details page, the profile completion is sixty percent. After the user fills all mandatory fields and clicks Continue or Save and Complete Later, the profile completion becomes seventy percent and the user is navigated to the Astrology page.

If the user clicks the Back Arrow, the user is navigated to the Hobbies and Interests page. The profile completion remains sixty percent. If the user did not previously save the data, the entered information is not stored.

The Continue button is enabled only when all mandatory fields are completed. Clicking Continue validates the required fields, saves the data, moves the user to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default. It becomes enabled only when all mandatory fields are entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.
