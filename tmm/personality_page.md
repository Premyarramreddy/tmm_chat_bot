Telugu Matchmakers Personality Page

The Personality page collects physical and lifestyle information of the bride or groom.

The following fields are mandatory:
Height
Weight
Diet
Smoking
Drinking
Disability Status

Blood Group is optional.

If the user selects Yes for Disability Status, Disability Details becomes mandatory.

When the user is on the Personality page, the profile completion is forty percent. After the user fills all mandatory fields and clicks Continue or Save and Complete Later, the profile completion becomes fifty percent and the user is navigated to the Hobbies and Interests page.

If the user clicks the Back Arrow, the user is navigated to the Profession and Education page. The profile completion remains forty percent. If the user did not previously save the data, the entered information is not stored.

The Continue button is enabled only when all mandatory fields are completed. Clicking Continue validates the required fields, saves the data, moves the user to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default. It becomes enabled only when all mandatory fields are entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.
