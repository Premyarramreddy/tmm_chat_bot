# Profile Verification Process – Telugu Matchmakers

## 1. Eligibility

* User can start profile verification only after completing profile setup.
* Verification can be initiated from the Profile Verification section.

---

## 2. Confirmation Details Screen

Before starting verification, the user will see the following details:

* First Name (Editable)
* Middle Name (Editable)
* Last Name (Editable)
* Gender (Not Editable)
* Date of Birth (Editable)
* Country (Editable)

### Important Guidelines

* First Name, Middle Name, Last Name, and Date of Birth must match the selected country’s government ID.
* The verification must be done using documents related to the selected country only.
* Users must ensure all details are accurate before proceeding.

---

## 3. Start Verification Flow

After confirming details, the user clicks **Continue** to start verification.

### Step 1: Select Country

Supported Countries:

* India
* USA
* UK
* Canada
* Australia
* New Zealand

### Step 2: Select Government ID Type

#### India

* PAN Card
* Aadhaar (National ID)
* Driving License
* Voter ID
* Passport

#### USA

* Driving License
* Visa
* Passport
* Green Card

(Other countries will show supported IDs accordingly.)

### Step 3: Upload Government ID

* User uploads a clear image of the selected ID.
* Document must be valid and not expired.
* All details must be readable.

### Step 4: Selfie Verification

* User takes a selfie following on-screen instructions.
* Ensure proper lighting.
* Face must be clearly visible.
* Avoid blurry or dark images.

---

## 4. Automated Verification Checks

The system performs automatic verification for:

* Document validity
* Expiry status
* Readability
* Face detection in selfie
* Document–Selfie match

### Possible Auto-Rejection Reasons

* Expired document
* Unreadable document
* Blurry image
* Face not detected
* Poor lighting

If auto-verification fails, the user will be prompted to retry.

---

## 5. Manual Verification (TMM Team Review)

After successful automated verification, documents are reviewed by the Telugu Matchmakers (TMM) team.

The team verifies whether confirmation details match the uploaded document.

### Manual Rejection Reasons

1. Name Mismatch
2. Date of Birth Mismatch
3. Country Mismatch

---

## 6. Rejection Handling & Attempts

### Attempt Limit

* Total allowed attempts: **3**

### Rejection Notification

When rejected, the user will see:

* Reason for rejection
* Attempt count (Example: `2/3 attempts left`, `1/3 attempts left`)

### Correction Screen

If rejected, the confirmation screen will display:

* First Name (as per document)
* Middle Name (as per document)
* Last Name (as per document)
* Date of Birth (as per document)
* Country (as per document)

Users must edit their details based on these values before retrying.

---

## 7. Re-Verification Process

After corrections, the user must repeat all steps:

1. Select Country
2. Select ID Type
3. Upload Document
4. Take Selfie

Each retry consumes one attempt.

---

## 8. Maximum Attempts Reached

If all 3 attempts fail:

* User will be blocked from further self-verification.
* A message will be shown to contact support.

### Support Options

* Email Support
* Phone Support

The TMM support team will review the case and resolve it within **24 hours**.

---

## 9. Best Practices for Users

* Use original government-issued documents.
* Ensure documents are not expired.
* Take photos in good lighting.
* Avoid reflections and shadows.
* Ensure details exactly match the document.

---

## 10. Verification Completion

If all checks pass:

* Profile status is updated to **Verified**.
* User gains access to all verified-user features.
* A success notification is displayed.
