Telugu Matchmakers About Page

The About page allows users to describe themselves and add a voice greeting to their profile.

The user must enter a profile summary. Voice greeting is optional. The user can choose a Telugu Matchmakers voice, record their own voice, or enhance the voice using AI. The user can also generate the profile summary using the AI generate feature.

When the user is on the About page, the profile completion is eighty percent. After the user fills the mandatory profile summary and clicks Continue or Save and Complete Later, the profile completion becomes ninety percent and the user is navigated to the Profile Picture page.

If the user clicks the Back Arrow, the user is navigated to the Astrology page. The profile completion remains eighty percent. If the data was not saved previously, the entered data is not stored.

The Continue button is enabled only when the profile summary is provided. Clicking Continue validates the required field, saves the data, moves the user forward, and increases the profile completion.

The Save and Complete Later button is disabled by default. It becomes enabled only when the profile summary is entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.
