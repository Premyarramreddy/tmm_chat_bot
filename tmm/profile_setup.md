Telugu Matchmakers Profile Setup Guide

Profile Setup is a step by step process that users complete after Sign Up or Sign In to create a full matrimonial profile. Each completed page increases the profile completion percentage and improves matchmaking.

Profile Setup Pages

The profile setup flow contains the following pages in order. Bride or Groom Name, Location, Religion and Community, Profession and Education, Personality, Hobbies and Interests, Family Details, Astrology, About, and Profile Picture.

Profile Completion Percentage

Each page contributes ten percent to the profile completion. Bride or Groom Name gives ten percent. Location gives twenty percent. Religion and Community gives thirty percent. Profession and Education gives forty percent. Personality gives fifty percent. Hobbies and Interests gives sixty percent. Family Details gives seventy percent. Astrology gives eighty percent. About gives ninety percent. Profile Picture gives one hundred percent.

If the account is created for Myself, the Bride or Groom Name page is skipped and profile setup starts directly from the Location page.

Common Page Behavior

Every profile setup page has a back arrow to go to the previous page, a continue button to save the current page and move to the next page, and a save and complete later option to save entered data and return to the Home Page.

If the user exits profile setup and comes back later, the process resumes from the last incomplete page.

Rules for Profile Setup

Each page contributes ten percent to completion. The completion percentage is based on pages, not on individual fields. Mandatory fields must be filled to continue to the next page. Optional fields can be skipped. The profile completion percentage is displayed on the Home Page until it reaches one hundred percent.
