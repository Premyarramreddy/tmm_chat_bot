Telugu Matchmakers Bride or Groom Name Page

The Bride or Groom Name page is shown only when the profile is created for someone other than Myself. If the account is created for Myself this page is skipped.

This page collects the first name, last name, email, and phone number of the bride or groom. First name, last name, and email are mandatory fields. Phone number is optional.

When the user is on this page the profile completion is zero percent. After the mandatory fields are filled and the user clicks Continue or Save and Complete Later, the profile completion becomes ten percent and the user is taken to the Location page.

If the user clicks the Back Arrow the user is taken to the previous screen which is either the onboarding or home page and the profile completion remains zero percent. If the data was not saved previously the entered data is not stored.

The Continue button is enabled only when all mandatory fields are filled. Clicking Continue validates the mandatory fields, saves the data, moves to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default. It becomes enabled only when all required details are entered. When clicked it saves the entered data, exits the profile setup flow, and redirects the user to the Home Page.
