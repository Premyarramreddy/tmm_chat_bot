Telugu Matchmakers Profession and Education Page

The Profession and Education page collects the career and education details of the bride or groom.

The following fields are mandatory:
Field of Profession
Profession Specialization
Income or Salary
Highest Qualification

The following fields are optional:
Organization
Field of Study
University or Institute

When the user is on the Profession and Education page, the profile completion is thirty percent. After the user fills all mandatory fields and clicks Continue or Save and Complete Later, the profile completion becomes forty percent and the user is navigated to the Personality page.

If the user clicks the Back Arrow, the user is navigated to the Religion and Community page. The profile completion remains thirty percent. If the user did not previously save the data, the entered information is not stored.

The Continue button is enabled only when all mandatory fields are completed. Clicking Continue validates the required fields, saves the data, moves the user to the next page, and increases the profile completion by ten percent.

The Save and Complete Later button is disabled by default and becomes enabled only when all mandatory fields are entered. When clicked, it saves the current page data, exits the profile setup flow, and redirects the user to the Home Page.
