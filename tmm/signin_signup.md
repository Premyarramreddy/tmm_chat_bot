Telugu Matchmakers Onboarding and Sign In Sign Up Guide

The Telugu Matchmakers application allows users to create an account or log in to an existing account through a simple onboarding process.

App Launch

When the application is opened, users are shown two options. One is Sign Up and the other is Sign In. Selecting either option will take the user to the corresponding screen.

Sign In

The Sign In feature is for users who already have an account.

To sign in, the user must enter their registered email ID or phone number and their password. Users may also enable biometric login by selecting the fingerprint checkbox if the device supports it. After entering the details, the user must click the Sign In button. If the credentials are valid, the user is taken to the Home Page.

Sign Up Entry

Users can start creating a new account by selecting Sign Up from the welcome screen or by clicking Join Now from the Sign In screen. Both options lead to the same Sign Up process.

Sign Up Step One Profile Basics

In the first step of Sign Up, users must select who the profile is being created for. The options are Myself, Brother, Sister, Son, Daughter, or Relative. The user must also select gender and date of birth.

The minimum age allowed to register is 21 years. If the entered age is less than 21, the user will not be allowed to continue. After entering valid details, the user clicks Continue to proceed.

Sign Up Step Two Personal and Account Details

In the second step, users must enter their first name, last name, email ID, mobile number based on the selected country code, and password.

The password must meet specific rules. It must contain at least one digit, one uppercase letter, one lowercase letter, and one special character. The password length must be between 8 and 16 characters.

Users can view the Terms and Conditions and Privacy Policy by clicking the provided links. They must accept both by selecting the checkbox before proceeding.

Verification Process

After clicking Continue, email verification and mobile number verification are started. A verification code is sent to the registered email ID and mobile number. Both verifications must be completed successfully.

Once both email and mobile number are verified, the account is created and the user is taken to the Home Page.

Onboarding Completion

The onboarding process is complete when the user successfully signs in or finishes the Sign Up process and reaches the Home Page. The user can then start using the Telugu Matchmakers application.
