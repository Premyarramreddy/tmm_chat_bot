Telugu Matchmakers Astrology Page

The Astrology page is completely optional. It allows users to provide horoscope related information if they wish.

The following fields are optional:
Zodiac Sign
Manglik Status
Star
Birth Time

No fields on this page are mandatory.

Both the Continue button and the Save and Complete Later button are enabled by default, even if no fields are filled.

If the user skips all fields, astrology details are not shown anywhere in the profile.

When the user is on the Astrology page, the profile completion is seventy percent. After the user clicks Continue or Save and Complete Later, the profile completion becomes eighty percent and the user is navigated to the About page.

If the user clicks the Back Arrow, the user is navigated to the Family Details page. The profile completion remains seventy percent. If the user did not previously save the data, the entered information is not stored.

Clicking Continue saves the current page data if any, moves the user to the next page, and increases the profile completion by ten percent.

Clicking Save and Complete Later saves the current page data if any, exits the profile setup flow, and redirects the user to the Home Page.
