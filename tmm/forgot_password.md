Telugu Matchmakers Forgot Password Guide

The Forgot Password option is available on the Sign In screen for users who cannot remember their password.

To reset the password, the user must click the Forgot Password link on the Sign In page. The user then enters the registered email ID or phone number. A six digit OTP is sent to the registered email ID. The user must enter the OTP for verification.

After OTP verification, the user must enter a new password and confirm the same password. The new password must follow specific rules. It must contain at least one digit, one uppercase letter, one lowercase letter, and one special character. The password length must be between eight and sixteen characters.

If the OTP is valid and the password meets all rules, the password is reset successfully. After a successful reset, the user is redirected to the Sign In page and can log in using the new password.
